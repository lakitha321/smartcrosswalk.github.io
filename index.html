Our Research

Our team is proposing to build up on an intelligent crosswalk system in order to provide a safer and more efficient pedestrian crossing experience by using Machine Learning , Deep Learning and Image Processing technologies.

Main components of our research :
  1. 1. Real-time Pedestrian Detection and Priority System for Disable-Pedestrian on Crosswalks
  2. 2. A real time traffic light system based on road condition, availability confirmation and uncontrollable speeds near crosswalks
  3. 3. Real-time Emergency Vehicle Detection and Priority System with Verification for Crosswalks
  4. 4. Pedestrian accident and rule violation detection on crosswalk

Our Research

Introduction

Current crosswalks system are inefficient so the probability of happening accidents are high and it does not provide a safe experience to the pedestrians.

Our team built up an intelligent crosswalk system in order to provide a safer and more efficient pedestrian crossing experience by using machine learning and image processing technologies.

Abstraction

The limitations of the traditional crosswalk system necessitate urgent reform for improved pedestrian safety and urban mobility. Inadequate pedestrian crossing times, prolonged waiting, emergency vehicle neglect, and a lack of 24/7 effective response mechanisms at traditional crosswalks pose pressing urban safety concerns. Our paramount objective is the creation of an advanced crosswalk system, underpinned by deep learning and image processing technologies. This research endeavors to innovate a sophisticated smart crosswalk system encompassing four key elements: a real-time Pedestrian Detection and Priority System tailored to special needs individuals, an adaptive vehicle availability and speed detective system in proximity to crosswalks, a real-time Emergency Vehicle Detection and Priority System fortified with rigorous verification protocols, and a robust framework for identifying pedestrian accidents and rule violations at crosswalks. The overall system is meticulously designed to not only enhance pedestrian safety by detecting potential hazards but also optimize traffic flow. In essence, it aspires to deliver a refined pedestrian crossing experience characterized by heightened safety and efficiency

Main Objective (Overall System)

Build an intelligent crosswalk system in order to provide a safer and more efficient pedestrian crossing experience by using machine learning and image processing technologies to detect potential hazards and improve traffic flow

Main Objective (Component 1)

Develop a pedestrian detection and time estimation model that can accurately count pedestrians and estimate the time duration for crossing the road, with a specific focus on prioritizing disabled pedestrians

Sub Objectives (Component 1)

Highlight the system's robust pedestrian detection and disabled pedestrian identification accurately

Emphasize the practical applications of the system for enhancing road safety, traffic flow, and urban development, while underlining the importance of real-world validation and future research directions

Main Objective (Component 2)

Create a dynamic traffic light control model that responds to real-world data, considering the conditions on the road, vehicle availability to reduce unnecessary idling time for pedestrians by training a machine learning model on image data derived from image processing

Sub Objectives (Component 2)

Implement a system at the road cross that accurately measures uncontrollable vehicle speed

Adjust traffic light state durations to ensure safe and efficient movement of both pedestrians and vehicles, to reduce the risk of accidents

Main Objective (Component 3)

Develop a smart crosswalk system to detect ambulances in real-time and prioritize their safe passage, thereby improving traffic safety and potentially saving lives

Sub Objectives (Component 3)

Detect the emergency vehicle siren sound using Iot to find the direction of the vehicle

Allow pedestrian to cross the road safely

Main Objective (Component 4)

Introduce a pedestrian accident detection model that utilizes image processing and machine learning techniques to accurately detect accidents on pedestrian crossings

Sub Objectives (Component 4)

Track vehicles that violate traffic rules by passing through crosswalks during red lights

Improve public safety by providing an effective pedestrian accident detection system

Inform authorities about the rule violations and accidents detected
